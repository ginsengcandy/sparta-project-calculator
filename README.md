<h1>1. 기본 정보</h1>
<h3> 1-1. 개요 </h3>
<h4>본 프로젝트는 Java의 객체지향적 특성을 활용하여 실제로 작동하는 기초적인 계산기 프로그램을 만들어보는 것에 의의가 있다.
본 프로젝트는 단계별로 3개의 서브 프로젝트로 구성되어 있으며, /src 폴더 안에 각각 step별로 구현되어 있다.</h4>

<h3> 1-2. 주요 기능</h3>
<h4>step1과 step2는 모두 사용자로부터 2개의 음이 아닌 정수와 1개의 계산 부호를 받아 두 수의 연산을 수행한다는 점에서 같다. 차이점은, 모든 기능이 main 메서드에 구현된 step1과 달리 step2는 연산 수행 기능이 독립된 Calculator 클래스 객체에서 이루어진다는 점이다.</h4>
<h4>이에 더해 step2에서는 연산 결과를 ArrayList 컬렉션에 저장하여 getter와 setter 메서드를 통해 누적된 연산 결과를 조회, 삭제(처음 저장한 인수 한정), 업데이트할 수 있다.

step3에서는 음이 아닌 정수뿐 아니라 실수도 받을 수 있으며, 컬렉션 조회 시 특정 값을 입력받아 해당 값보다 더 큰 값만 조회하는 기능을 가진 .getBiggerItem() 메서드를 추가로 구현했다.
</h4>

<h3> 1-3. 기술 스택</h3>
<h4> - Step1, Step2, Step3 공통 : 사용자로부터 숫자와 문자열을 입력받기 위해 Scanner 클래스를 import했다.
</h4>
<h4> - Step2 : 사용자로부터 입력받는 타입이 이를 저장하는 변수의 타입과 일치하지 않을 때 발생하는 InputMismatchException을 처리하기 위해 해당 예외 클래스를 import했다.
</h4>

<h1> 2. 배운 점</h1>
<h4>향후 계획</h4>
앞으로 단계별로 브랜치를 만들어야겠다. step2를 하다가 step1을 수정하면 커밋 시 step1과 step2의 수정사항 모두를 기입해야 해 복잡해진다.
